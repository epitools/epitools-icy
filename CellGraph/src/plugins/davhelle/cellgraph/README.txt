 Welcome to CellGraph, a graph based plugin for the imaging software ICY 
 
 What is currently needed to run it
 1. Obtain icy: an open community platform for bioimage informatics
 	link: http://icy.bioimageanalysis.org
 2. Obtain JTS: the Java topology suite 
 	link: http://sourceforge.net/projects/jts-topo-suite/
 3. Obtain JGraphT: a free Java graph library
 	link: http://sourceforge.net/projects/jgrapht/
